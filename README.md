# Simple GUI for NordVPN (Linux)
A no-frills, simple GUI for NordVPN on Linux using only Python's built-in modules.

Work-in-progress.

### TEST RIG SPECS:
Linux:
1) OS: 64-bit Debian 12 (Bookworm)
2) System: Raspberry Pi 5
3) CPU: Broadcom BCM2712 (4 cores @ 2.4GHz)
4) RAM: 8GB LPDDR4X
5) GPU: VideoCore VII
